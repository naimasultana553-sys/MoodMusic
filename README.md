# Mood-Based Music Recommendation System

A modern, responsive web application that uses a Machine Learning model to predict a user's mood from text and recommends songs based on that mood.

## Features

- **Mood Prediction**: Text analysis to predict user mood.
- **Dynamic Theming**: The UI changes accent colors based on the predicted mood.
- **Glassmorphism Design**: Modern UI with beautiful glass-like cards.
- **Song Recommendations**: Recommends songs matching the user's emotion.
- **Responsive**: Works perfectly on Desktop, Tablet, and Mobile.
- **Dark/Light Mode**: Toggleable themes.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Backend**: Python, Flask

## Setup

1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python app.py
   ```
3. Open `http://localhost:5000` in your browser.
